# MusicGrid

**Create beats with multiple tracks and samples.**

This is a web app where you can make music using a step sequencer. You can pick different sounds, set the number of beats and measures, and play your rhythm in real time.
Check out the [video](https://youtu.be/JS07E6a26bM) for more understanding.

## Features

- Pick from multiple sound samples for each track.  
- Set how many measures and beats you want.  
- Click steps to make sounds play in order.  
- See which steps are active while playing.  
- Works on mobile, tablet, and desktop.  
- Light and dark mode toggle.  

## Self Hosting Guide

1. Clone the repository:  
   ```bash
   git clone https://github.com/avalynndev/grid
   ```

2. Install dependencies:
    ```
    npm install
    ```

3. Run the project:
    ```
    npm run dev
    ```

Open `http://localhost:3000` in your browser.

Pick samples, add steps, and start making beats.
